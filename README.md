### Windows下快捷键
* 截图工具 
  * win + shift + s
* Chrome浏览器切前一个标签页
  * ctrl + shift + tab
  * ctrl + pgup
* Chrome浏览器切后一个标签页
  * ctrl + tab
  * ctrl + pgdn
* Chrome浏览器指定标签页
  * ctrl + 数字
* Chrome浏览器切最后一个标签页
  * ctrl + 9
* 切页面
  * alt + tab
